# Ads Campaign Dashboard

A responsive, feature-rich dashboard for managing and analyzing advertising campaigns. Built with modern web technologies to ensure performance and a great user experience.

## Tech Stack

-   **Framework**: [Next.js](https://nextjs.org/)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
-   **Charts**: [Recharts](https://recharts.org/)
-   **State Management**: [TanStack Query](https://tanstack.com/query/latest)

## Features

-   **Campaign Management**: View, sort, and filter campaigns.
-   **Detailed Insights**: Analyze campaign performance with interactive charts.
-   **Real-time Updates**: Live stream of campaign insights.
-   **Responsive Design**: distinct mobile and desktop experiences.
-   **Dark/Light Mode**: Fully themeable UI.

## Environment Variables

To run this project, you will need to add the following environment variables to your `.env` file:

`NEXT_PUBLIC_API_URL`= https://mixo-fe-backend-task.vercel.app

## Getting Started

1.  **Install dependencies**:
    ```bash
    pnpm install
    ```

2.  **Run the development server**:
    ```bash
    pnpm dev
    ```
